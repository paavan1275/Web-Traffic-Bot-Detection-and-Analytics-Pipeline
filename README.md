# Web-Traffic-Bot-Detection-and-Analytics-Pipeline
This project detects bots in web traffic using session‑level features engineered from web server logs.

## Overview
Parse web logs and group requests into sessions

Build behavioural and semantic features for each session

Train and compare models (Random Forest, XGBoost, LightGBM)

Select Random Forest as the final model based on best performance and stability

## Tech Stack
Python, pandas, NumPy

scikit‑learn (RandomForest), XGBoost, LightGBM
